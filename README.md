_A video library web application built using MERN stack._

## Installation and Setup Instructions

Clone this repository. You will need `node` and `npm` installed globally on your machine.

Installation:

```bash
npm install
```

To Start Server:

```bash
npm start
```

Visit app at:

`localhost:3000`
